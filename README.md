# CTC to In-Hand Salary Calculator (India)

This is a free online tool to calculate monthly in-hand salary from annual CTC in India.

## Features
- CTC to in-hand salary calculation
- PF and tax estimation
- New tax regime
- Mobile-friendly
- Fast and lightweight

https://yashwantsirsate60.github.io/ctc-to-inhand-salary-calculator/
